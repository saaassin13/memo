# 备忘录页面详细设计 (Step 4)

## 页面概述

| 页面 | 路由 | 说明 |
|------|------|------|
| 备忘录列表 | `/memo` | 展示所有备忘录，支持分类筛选 |
| 新建备忘录 | `/memo/edit` | 新建备忘录 |
| 编辑备忘录 | `/memo/edit?id=1` | 编辑已有备忘录 |

---

## 页面一：备忘录列表

### 1.1 视觉设计

#### 整体风格
- **设计语言**：Material Design 3 + 圆角卡片
- **圆角**：16dp（卡片）、12dp（标签）、24dp（FAB）
- **间距**：16dp（页面边距）、12dp（卡片间距）

#### 配色方案
| 元素 | 颜色 |
|------|------|
| 页面背景 | `colorScheme.surface` |
| 卡片背景 | `colorScheme.surfaceContainerHighest` |
| 主色调 | `#667EEA` (紫蓝渐变) |
| 置顶标记 | `#FFB800` (金色) |
| 文字标题 | `colorScheme.onSurface` |
| 文字摘要 | `colorScheme.onSurfaceVariant` |
| 文字日期 | `colorScheme.outline` |

#### 分类标签颜色
| 分类 | 颜色 |
|------|------|
| 全部 | `#667EEA` |
| 工作 | `#4F46E5` |
| 生活 | `#10B981` |
| 学习 | `#F59E0B` |
| 杂项 | `#6B7280` |

### 1.2 页面结构

```
┌─────────────────────────────────────┐
│  ← 备忘录                    🔍  ⋮  │  <- 顶部导航栏 (56dp)
├─────────────────────────────────────┤
│ [全部] [工作] [生活] [学习] [杂项]   │  <- 分类标签栏 (48dp)
├─────────────────────────────────────┤
│                                     │
│  ┌──────────────┐ ┌──────────────┐ │
│  │ 📌 会议记录   │ │ 购物清单      │ │  <- 2列网格卡片
│  │ 今天下午三... │ │ 需要买牛...   │ │
│  │ 01-15 14:30  │ │ 01-14        │ │
│  └──────────────┘ └──────────────┘ │
│                                     │
│  ┌──────────────┐ ┌──────────────┐ │
│  │ 学习笔记      │ │ 电影清单      │ │
│  │ 今天要看...   │ │ 周末看...     │ │
│  │ 01-13        │ │ 01-12        │ │
│  └──────────────┘ └──────────────┘ │
│                                     │
│  ... 更多列表项 ...                 │  <- 可滚动网格
│                                     │
│                              [+]    │  <- 悬浮按钮 (56dp)
└─────────────────────────────────────┘
```

### 1.3 组件设计

#### 顶部导航栏
- **高度**：56dp
- **左侧**：返回箭头（从子页面返回时显示）
- **中间**：标题「备忘录」
- **右侧**：搜索图标、更多菜单（⋮）
  - 搜索：展开搜索框
  - 更多：排序方式（最新/最早/置顶优先）

#### 分类标签栏
- **高度**：48dp
- **布局**：横向滚动，单选模式
- **标签样式**：
  - 未选中：透明背景 + 灰色文字
  - 选中：圆角背景（12dp）+ 主色文字 + 主色背景 10% 透明度
- **点击效果**：150ms 缩放动画

#### 备忘录卡片（2列网格）
- **布局**：2列网格，SliverGrid
- **卡片高度**：120dp
- **内边距**：12dp
- **结构**：
  - 第一行：置顶图标（如果有）+ 标题（最多1行，溢出省略）
  - 第二行：摘要（最多2行，溢出省略）
  - 第三行：日期
- **置顶样式**：左上角金色置顶图标
- **点击效果**：波纹动画 + 150ms 缩放（0.98）
- **长按**：弹出操作菜单（编辑/删除/置顶）

#### 悬浮按钮 (FAB)
- **位置**：右下角，距边缘 16dp
- **尺寸**：56dp × 56dp
- **图标**：加号 (+)
- **样式**：主色渐变背景，白色图标，阴影
- **点击效果**：波纹 + 缩放动画

### 1.4 交互设计

#### 页面切换动画
- **进入**：从右侧滑入（300ms，ease-out）
- **退出**：向右侧滑出（300ms，ease-in）
- **共享元素**：无

#### 列表交互
- **下拉刷新**：刷新列表数据
- **上拉加载**：分页加载更多
- **滑动操作**：
  - 左滑：显示删除按钮（红色）
  - 右滑：显示编辑按钮（蓝色）
- **空状态**：显示插画 + "暂无备忘录，点击 + 新建"

#### 搜索交互
- **触发**：点击搜索图标
- **效果**：标题栏展开为搜索框
- **搜索**：实时过滤（300ms 防抖）
- **关闭**：点击空白区域或返回键

## 页面二：新建/编辑备忘录

### 2.1 视觉设计

#### 配色方案
| 元素 | 颜色 |
|------|------|
| 页面背景 | `colorScheme.surface` |
| 输入框背景 | `colorScheme.surfaceContainerHighest` |
| 主色调 | `#667EEA` |
| 分类标签 | 参照上方分类颜色 |
| 警告色 | `#EF4444` (删除) |

### 2.2 页面结构

```
┌─────────────────────────────────────┐
│  ←  新建备忘录              保存    │  <- 顶部导航栏
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐   │
│  │ 输入标题...                   │   │  <- 标题输入
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │ 输入内容...                   │   │  <- 内容输入
│  │                             │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
├─────────────────────────────────────┤
│  分类: [工作] [生活] [学习] [杂项]   │  <- 分类选择
├─────────────────────────────────────┤
│  ⏰ 提醒时间: [点击设置]             │  <- 提醒设置
├─────────────────────────────────────┤
│  📌 置顶: [开关]                     │  <- 置顶开关
└─────────────────────────────────────┘
```

### 2.3 组件设计

#### 标题输入框
- **高度**：56dp
- **字体**：20sp，加粗
- **占位符**：「输入标题...」
- **最大长度**：100字符

#### 内容输入框
- **高度**：自适应，最小 200dp
- **字体**：16sp，正常
- **占位符**：「输入内容...」
- **特性**：支持多行自动扩展

#### 分类选择
- **布局**：横向排列，单选
- **样式**：Chip 组件
- **默认**：无选中

#### 提醒时间
- **样式**：ListTile
- **点击**：弹出日期时间选择器
- **显示**：格式「YYYY-MM-DD HH:mm」
- **清除**：点击清除按钮

#### 置顶开关
- **样式**：Switch 组件
- **位置**：ListTile trailing

### 2.4 交互设计

#### 页面切换动画
- **进入**：从底部滑入（300ms，ease-out）
- **退出**：向底部滑出（300ms，ease-in）

#### 保存逻辑
- **自动保存**：输入后 2 秒自动保存到草稿
- **手动保存**：点击保存按钮
- **验证**：标题不能为空
- **保存后**：返回列表页，刷新数据

#### 返回确认
- **触发**：返回键 + 未保存更改
- **弹窗**：「是否保存更改？」
  - 保存：保存并返回
  - 不保存：丢弃并返回
  - 取消：留在当前页面

## 状态设计

### 列表页状态
| 状态 | 显示 |
|------|------|
| 加载中 | 骨架屏 |
| 空列表 | 空状态插画 |
| 加载错误 | 错误提示 + 重试按钮 |
| 正常 | 列表数据 |

### 编辑页状态
| 状态 | 显示 |
|------|------|
| 新建 | 空表单 |
| 编辑 | 填充已有数据 |
| 保存中 | 按钮 loading |
| 保存成功 | 提示 + 返回 |
| 保存失败 | 错误提示 |

## 数据模型

```dart
class Memo {
  final int? id;
  final String title;           // 标题 (必填, max:100)
  final String content;         // 内容
  final String? category;       // 分类: 工作/生活/学习/杂项
  final bool isPinned;          // 是否置顶
  final DateTime? remindTime;   // 提醒时间
  final DateTime createdAt;     // 创建时间
  final DateTime updatedAt;     // 更新时间
}
```
