# 日历页面设计 (Step 8)

## 概述

日历页面是底部导航的第三个 Tab，提供日/周/月三种视图模式，展示各类时间相关事件和信息。包括备忘、待办、纪念日、目标进度、体重记录等。

---

## 模块设计

### 1. 顶部模式切换

| 元素 | 功能 |
|------|------|
| 日视图 | 垂直时间轴，每小时一个格子 |
| 周视图 | 7列布局，每天一个竖向列表 |
| 月视图 | 传统月历格子，有事件标记点 |

**UI 设计**:
- 三个胶囊按钮横向排列
- 选中态：渐变背景 + 白色文字
- 未选中态：灰色背景 + 灰色文字
- 切换动画：300ms ease-in-out

### 2. 月份导航

| 元素 | 功能 |
|------|------|
| 左箭头 | 上一个月 |
| 年月标题 | 显示「2024年1月」 |
| 右箭头 | 下一个月 |
| 今天按钮 | 快速回到今天 |

**UI 设计**:
- 标题使用大号字体 (20sp, fontWeight: 700)
- 箭头按钮圆形背景
- 今天按钮文字按钮样式

## 数据模型

### CalendarEvent (日历事件)

```dart
class CalendarEvent {
  final String id;
  final CalendarEventType type; // 备忘/待办/纪念日/目标/体重
  final DateTime date;
  final String title;
  final String? description;
  final int? targetId;      // 关联目标ID
  final double? weight;     // 体重记录
  final int? daysUntil;     // 纪念日倒计时
}
```

### CalendarEventType (事件类型)

```dart
enum CalendarEventType {
  memo,      // 备忘
  todo,      // 待办
  anniversary, // 纪念日
  goal,      // 目标
  weight,    // 体重
}
```

## 状态管理

### Providers

| Provider | 类型 | 功能 |
|----------|------|------|
| calendarViewModeProvider | StateProvider | 日/周/月模式 |
| selectedDateProvider | StateProvider | 选中日期 |
| calendarEventsProvider | FutureProvider | 加载事件数据 |
| eventTypeFilterProvider | StateProvider | 事件类型筛选 |

### 事件加载逻辑

1. 根据选中日期加载当月数据
2. 按类型筛选显示
3. 月视图: 聚合每天事件
4. 周视图: 聚合每周事件
5. 日视图: 按时间排序

## 注意事项

1. **性能优化**: 月视图滚动时只渲染可见区域
2. **数据缓存**: 事件数据本地缓存，减少重复请求
3. **空状态**: 无事件时显示友好提示
4. **日期边界**: 处理跨月、跨年日期显示
5. **动画流畅**: 视图切换使用 300ms 动画
